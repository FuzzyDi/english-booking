# Детский сайт записи на уроки английского

## Запуск
1. Установите Python 3
2. Выполните:
   pip install flask
   python app.py
3. Откройте http://localhost:5000

## Админка
- URL: /admin
- Пароль: teacher123

## Особенности
- Только имя + телефон
- 1 запись в день, макс. 3 в неделю
- Преподаватель может отключать слоты
- Все данные хранятся в database.db
